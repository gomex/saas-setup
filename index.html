Introduction

"DevOps" Setup Guide

This guide covers the basics of setting up an environment to work with SaaS/"DevOps"/SRE role. Whether you are an experienced programmer or not, this guide is intended for everyone to use as a reference for setting up your environment. For now we are focus on Mac, but we accept PR.

Contributing to the guide

All contributions to the guide are welcome. Please help add support for other libraries and languages. To make a contribution please use our contribution template.

We're looking for more contributors to maintain and extend the documentation.
